# 📶 Wifi Ninja

**Wifi Ninja** is a powerful Android utility app designed for advanced users. With root access, you can manage saved Wi-Fi networks like a true ninja — view, extract, share, back up, and restore Wi-Fi credentials with ease.

---

## 🔥 Features

- ✅ View and extract saved Wi-Fi passwords (requires root)
- 📤 Share Wi-Fi credentials via:
    - QR Code
    - Plain text
- 💾 Backup saved Wi-Fi networks
- ♻️ Restore Wi-Fi networks from backup
- 🌈 Supports Android 5.0 (Lollipop) through Android 15

---

## ⚙️ Requirements

- 📱 Android 5.0 – Android 15
- 🔓 **Root access is required**
